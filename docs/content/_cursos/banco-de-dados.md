Se você quer entender como os dados são organizados, consultados e protegidos — e, de quebra, executar consultas reais com SQL — você está no lugar certo. Este curso é uma jornada prática pelos bastidores dos sistemas que guardam informações do mundo digital.

Aqui a gente aprende na prática: nada de respostas prontas — vamos modelar, normalizar, escrever consultas e integrar bancos de dados em aplicações. O objetivo é te deixar confortável para projetar e usar bancos de dados relacionais em projetos reais.

O que você vai conseguir no final deste curso?

- Modelar dados com diagramas (DER) e transformar modelos conceituais em modelos lógicos.
- Projetar esquemas normalizados que evitem redundância e garantam integridade.
- Escrever consultas SQL para criar, manipular e consultar dados.
- Entender transações, concorrência e noções de segurança e desempenho.
- Ter uma visão introdutória sobre alternativas NoSQL e quando usá-las.

Para quem é este curso?

- Estudantes e profissionais de Engenharia de Sistemas, Ciência da Computação e áreas relacionadas.
- Pessoas curiosas que querem entender como os sistemas armazenam e recuperam informação.
- Não é obrigatório saber programar profundamente, mas alguma familiaridade com lógica e estruturas de dados ajuda.

Formato do curso

- Aulas teóricas curtas intercaladas com exercícios práticos.
- Laboratórios com exemplos em SQL e pequenos projetos de modelagem.
- Estudos de caso para conectar teoria e prática.

Resumo do objetivo

Capacitar você a compreender os fundamentos teóricos e práticos de sistemas de banco de dados, habilitando-o a modelar, projetar, implementar e manipular bancos de dados relacionais — com ênfase na aplicação prática em engenharia de sistemas.

Próximo passo: mãos à obra! Vamos começar pela modelagem conceitual (DER) e entender como transformar requisitos em tabelas bem desenhadas.
