## CONCEITO

"Um Sistema Gerenciador de Banco de Dados (SGBD) é constituído por um conjunto de dados associados a um conjunto de programas para acesso a esses dados."

"Os SGBDs são sistemas responsáveis em armazenar os dados e fornecer serviços capazes de manipulá-los. Para armazenar os dados em um disco, os SGBDs possuem uma forma estruturada padrão de representar os registros e campos, fornecendo serviços que permitam a criação e alteração dessas definições de dados. Fornecem ainda, um mecanismo interno para manter os dados no disco e para saber onde está cada elemento em particular."

## SGBD x SO

Em alguns casos podemos comparar um SGBD com um Sistema Operacional (SO), uma vez que o mesmo pode ter as mesmas funções:
- Controle de processos
- Controle de escrita e leitura em disco
- Segurança de acesso ao dados

---

## CARACTERÍSTICAS

- **Controle centralizado**
- **Controle de concorrência**: Acesso simultâneo de vários usuários mantendo a consistência dos dados
- **Recuperação de falhas**
- **Segurança das informações**: Acesso e integridade física

---

## CARACTERÍSTICAS (CONTINUAÇÃO)

- Facilita administração centralizada por um DBA – Administrador de banco de dados
- Pode reduzir a redundância
- Inconsistência pode ser evitada
- Pode compartilhar os dados
- Pode aplicar restrições de segurança
- Pode manter a integridade
- Pode equilibrar as necessidades conflitantes
